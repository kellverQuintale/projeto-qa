# Evidências de Testes

Esta pasta contém evidências das execuções de testes realizadas durante a validação das funcionalidades.

## Tipos de evidências

- Prints de execução dos testes
- Evidências de bugs encontrados
- Validações de cenários positivos e negativos

## Organização

As evidências podem ser separadas por funcionalidade:

Exemplo:

📁 Login  
- login_sucesso.png
- login_erro_senha.png

📁 Cadastro  
- cadastro_valido.png
- cadastro_campo_obrigatorio.png
